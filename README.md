## **Номер 1**

**Название:** «Ввод валидного email-адреса»

**Шаги воспроизведения:**

1. Ввести в поле ввода email «[skibidi@gmail.com](mailto:skibidi@gmail.com)».
2. Нажать кнопку «Проверить».

**Ожидаемый результат:** появился alert с текстом «Хороший email-адрес».

---

## **Номер 2**

**Название:** «Отправка пустого input»

**Шаги воспроизведения:**
